# BingoGame
Bingo Game By Nitin

To Run:
## npm start

To Play:

1. Step 1:
  * Admin Logs-in to Admin Panel.
2. Step 2:
  * Multiple Playes Logs into Game with Nickname.
3. Step 3:
  * Everyone Choose Game Size Appropriately (Same Size).
4. Step 4:
  * Admin Generaes and Sends Numbers

Enjoy Your Bingo Game.
